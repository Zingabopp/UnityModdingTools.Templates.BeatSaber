This extension should not be installed with the BeatSaberModdingTools extension (the templates will conflict).

Contains the following templates:

Project Templates
* BSIPA4 Plugin (Bare)
* BSIPA4 Plugin (Core)
* BSIPA4 Plugin (Disableable)

Item Templates
* BSML File
* BSML ViewController
* BSML ViewController (Bare)
* Harmony Patch
* Harmony Patch (Bare)
* MonoBehaviour
* MonoBehaviour (Bare)
* Zenject Installer
